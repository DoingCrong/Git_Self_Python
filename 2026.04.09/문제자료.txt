|필드명                 | 한글      | 설명                      |
| ---------------       | -----      | ----------------------- |
| date                  | 날짜       | 매출이 발생한 날짜              |
| year                  | 연도       | 해당 데이터의 연도              |
| month               | 월         | 해당 데이터의 월               |
| quarter              | 분기       | 1~4분기 (예: 1분기, 2분기)     |
| region               | 지역       | 매출이 발생한 지역              |
| product_line       | 제품군     | 제품 또는 서비스 종류            |
| units_sold          | 판매수량  | 판매된 개수                  |
| unit_price          | 단가        | 제품 1개의 가격               |
| discount_rate      | 할인율     | 할인 적용 비율 (예: 0.1 = 10%) |
| marketing_spend | 마케팅비  | 광고/홍보에 사용된 비용           |
| revenue         | 매출액        | 총 매출 금액                 |


메뉴
홈 / 연도별 매출 /지역별 매출 / 제품별 수익성

매출 = 판매수량 * 단가 * (1-할인율)
제품별 수익성 : 매출액 - 마케팅비
연도별 매출 : 해마다 매출이 얼마나 증가했는지 확인
지역별 매출 : 어느 지역의 매출이 높은지 확인
제품 수익성 : 어떤 제품의 수익성이 좋은지 확인
