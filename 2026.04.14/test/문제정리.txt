포스트맨(이름, 국어, 영어, 수학)으로 점수 데이터를 보내면 서버에서 총점, 평균을 계산하고, 웹 화면에 
이름별 총점과 전체 평균선을 차트로 보이도록 작성

프로그램명
Flask + JSON + Chart.js를 이용한 성적 처리 프로그램

요구사항

포스트맨에서 다음 데이터를 JSON 형식으로 입력받는다.

이름
국어
영어
수학

입력받은 자료를 이용하여 다음을 계산한다.

총점
평균

처리된 결과는 서버에 저장한다.

웹 브라우저에서 Chart.js를 이용하여 다음 차트를 출력한다.

x축: 이름
y축: 총점
이름별 총점 막대그래프
전체 총점 평균값을 기준선으로 표시

사용 기술

Flask
JSON
Chart.js

1. 포스트맨에서 이름, 국어, 영어, 수학을 JSON으로 전송
2. Flask가 총점, 평균을 계산
3. 계산 결과를 메모리에 저장
4. chart페이지에서 chart.js로 차트 출력
    이름별 총점 막대그래프
    전체 총점 평균선
